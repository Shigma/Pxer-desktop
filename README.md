# Pxscope

一个 Pixiv 的桌面版。

## 项目说明

文件目录：
```
assets 存放字体，图标等资源
build 存放打包所用的代码
comp 存放各个页面组件
dist 存放打包后的文件
i18n 存放多语言配置
themes 存放多主题配置

main.js 主程序入口
index.html 主界面入口
app.js 视图模型入口
```

## 本地构建

1. clone 这个项目。
2. `git submodule init`初始化子模块。
3. `npm install`更新依赖。
4. `npm run build`进行构建。
5. `npm run start`开始运行。
